# 2026-06-19（五）模拟面试闭环 D1：设计 + 出题

## 目标

模拟面试子图立起来：能基于备战计划和知识库出题，CLI 里能开始一轮问答。

## 设计（先定再写）

- [ ] 30 分钟设计文档（写 `docs/mock_interview.md`，半页即可）：
  - 状态：`InterviewState(plan, history[(question, answer, score, feedback)],
    current_question, weak_areas, round_count, max_rounds)`
  - 图：`pick_topic → ask → (等待用户作答) → score → feedback → 路由(继续/收尾) → summary`
  - 关键决策：人机交互用 LangGraph `interrupt`（checkpointer + Command(resume) 续跑，
    这是 LangGraph human-in-the-loop 的标准姿势，也是简历亮点）还是
    简单的"图外循环、每轮 invoke 一次"。**评估后定，复杂度失控就选后者。**

## 任务
